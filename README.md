# Car-Simulation-GRAPHICS
2D Car Simulation Project

A simple and interactive 2D Car Racing Simulation Game built using Python and Pygame.
The player controls a blue car, avoids enemy cars, scores points, and survives as long as possible.

Features:
Smooth car movement using keyboard controls
Multiple enemy cars with random speed and positions
Collision detection system
Score tracking system
Pause and Resume functionality
Restart option after Game Over
Animated road divider effect
Simple and clean UI design
Technologies Used:
Python
Pygame
